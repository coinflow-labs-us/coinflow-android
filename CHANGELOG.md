# Changelog

## 0.1.0

- Initial release of `cash.coinflow:coinflow-card-form`.
- WebView-backed card tokenization with `CARD_FORM`, `CARD_NUMBER_FORM`, and `CVV_FORM` variants.
- `MerchantTheme` support for styling the rendered form.
- `CoinflowCardForm.tokenize()` returns a `CardFormTokenResponse` with token and optional expiry.
